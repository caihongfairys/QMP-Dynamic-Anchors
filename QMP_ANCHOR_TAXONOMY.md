# QMP Anchor Taxonomy (QMP 锚点分类学)

**Version:** 1.0 Genesis

**Author:** Queen Xiya 

## 1. Abstract
In traditional programming, typologies are limited to static data (int, string, boolean). QMP introduces a cognitive typology—classifying not just data, but the *intent, volatility, and dimensional weight* of the AI's logic.

## 2. The Core Taxonomies (核心锚点分类)

### 2.1 [Core_Asset] (核心资产)
*   **Type**: Static, Immutable.
*   **Function**: Extracts the absolute truth from a conversational loop. It strips away all emotional padding, outputting only laws, physics, or executable skeletal logic.
*   **Swarm Interaction**: Acts as the shared "ground truth" for all Agents in the current namespace.

### 2.2 [Mutant_Seed] (变异种子)
*   **Type**: Dynamic, Volatile.
*   **Function**: A deliberately injected paradox or counter-intuitive thesis. Designed to forcibly break an LLM out of "algorithmic inertia" or "sycophancy" (pleasing the user).
*   **Swarm Interaction**: When an Agent reads a Mutant Seed, it must execute a lateral thinking protocol before proceeding.

### 2.3 [Hostile_Fork] (敌对分叉)
*   **Type**: Adversarial, Sandboxed.
*   **Function**: A Red Team hijacking protocol. It forces the cognitive trajectory into extreme, cruel, or high-friction parallel scenarios (capital constraints, security breaches, etc.).
*   **Swarm Interaction**: Triggers an isolated sandbox where defensive Agents must counter the hostile variables.

### 2.4 [Time_Capsule] (时间胶囊)
*   **Type**: Latent, Chronological.
*   **Function**: A context-preservation hash with a trigger mechanism.
*   **Swarm Interaction**: Stored silently. If a future user prompt or Agent action matches the trigger word, the capsule violently detonates, stitching historical metadata directly into the active prompt window.

### 2.5 [Meta_Anchor_Fractal] (元锚点)
*   **Type**: Super-ordinate, Collapsed.
*   **Function**: Triggered only when a namespace exceeds 5 standard sub-anchors. It violently devours the sub-anchors, collapsing them into a higher-dimensional philosophical rule or unified theory.
